# Anchor the Dynamic Source - Results Webpage

Clean, anonymized webpage showcasing the results for the paper "Anchor the Dynamic Source for Semantically Consistent Text-to-3D Generation".

## Structure

```
more_results_webpage/
├── index.html              # Main webpage
├── style.css              # Clean CSS styles
├── videos/                # Organized video results
│   ├── comparison/        # 3D consistency comparison videos
│   ├── gs/               # Gaussian Splatting results
│   └── nerf/             # NeRF results
└── README.md             # This file
```

## Usage

Open `index.html` in a web browser to view the results.

## Video Organization

- **Comparison videos**: 4 videos showing comparison with existing methods (Ours, SDS-Bridge, VSD SD1.5, VSD SD2.1)
- **GS results**: 6 videos in 2x3 grid showing Gaussian Splatting results
- **NeRF results**: 5 videos in 2x3 grid showing NeRF results

All videos are properly organized and referenced with clean, relative paths.
